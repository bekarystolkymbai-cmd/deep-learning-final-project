Project Proposal: Plant Disease Classification 1. Project OverviewThe goal of this project is to develop a Deep Learning model capable of identifying plant diseases from leaf images. Early detection of plant pathology is critical for food security and minimizing economic losses for farmers.2. Dataset DescriptionName: PlantVillage DatasetSource: Kaggle / PlantVillageDetails: The dataset contains approximately 54,305 images of plant leaves categorized into 38 classes (including healthy and diseased states across 14 plant species like Tomato, Potato, Apple, and Grape).Input: RGB Images (varying sizes, to be resized to $224 \times 224$ or $299 \times 299$).3. Planned Deep Learning ApproachBaseline Model: A custom Convolutional Neural Network (CNN) built from scratch to establish a performance floor.Advanced Model: Transfer Learning using architectures such as ResNet50 or EfficientNetB0 pre-trained on ImageNet.Optimization Techniques: * Data Augmentation: Horizontal/Vertical flips, rotation, and brightness adjustments to improve generalization.Regularization: Dropout and Batch Normalization.LR Scheduler: Reducing the learning rate when validation loss plateaus.4. Evaluation MetricsPrimary: Accuracy Score.Secondary: F1-Score (to account for potential class imbalances) and Confusion Matrix (to identify which diseases the model confuses most often).
